## Basic
- `Python` >= 3.8
## Modules

- `h5py`
- `numpy`
- `matplotlib` <= 3.5.1
- `rasterio`
- `tensorflow-gpu` >= 2.5.0
- `pandas`
- `sklearn`
- `argparse`
- `PIL`
- `tifffile`
- `seaborn`
- `shutil`

pip install -r requirements.txt
